Use WooMultistore and WP All Import to import your products to several WooCommerce stores at once.

== Description ==
This is a free WooCommerce WooMultistore Add-On for WP All Import that makes you able to use WP All Import to import your products to several WooCommerce stores at once.

This addon requires the [WooMultistore plugin](https://woomultistore.com), [WP All Import](http://www.wpallimport.com/) + the [WP All Import Woocommerce Product Import Add-On](http://www.wpallimport.com/woocommerce-product-import/).
